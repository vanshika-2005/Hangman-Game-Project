# Hangman Game

A simple Hangman game implemented in Python. This game can be played in single-player or multiplayer mode, where players try to guess a secret word by suggesting letters within a limited number of attempts.

## Features

- Single-player and multiplayer modes available.
- Multiple themes to choose from for the secret word.
- Three levels of difficulty: easy, medium, and hard.
- Option to receive hints in hard difficulty.
- ASCII art hangman visualization for each incorrect guess.
- Option to play again after the game ends.

## Usage

1. Clone the repository:

```bash
git clone https://github.com/your-username/hangman-game.git
```

2. Navigate to the directory:

```bash
cd hangman-game
```

3. Run the Python script:

```bash
python hangman.py
```

4. Follow the on-screen instructions to play the game.

## Requirements 

Python 3.x

## Gameplay

- Choose the game mode: single player or multiplayer.
- Select a theme for the secret word: cars, food, colors, cartoons, or mixed.
- If playing in single-player mode, choose the difficulty level.
- Guess letters to reveal the secret word.
- For multiplayer mode, take turns guessing letters.
- Receive hints in hard difficulty if needed.
- Win by guessing the word correctly or lose by running out of guesses.
